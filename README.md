## How I think about agentic AI

- An LLM that only answers questions is a very expensive analyst who never gets to implement anything. The value is realized when it becomes part of a **system that takes action**.
- Regulated industries are not forgiving environments for brittle automation — design for **human-in-the-loop escalation** from the start, not as an afterthought.
- The governance layer isn't overhead. In regulated industries, **the audit trail is the product.**
